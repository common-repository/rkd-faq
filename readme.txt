== Description ==

Creates a FAQ Accordion

== Installation ==

1. Upload the files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter questions/answers through the admin panel
4. Use the shorttag [rkdfaq] on a page
